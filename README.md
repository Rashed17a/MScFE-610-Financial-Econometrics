## 📁 Project Descriptions

### 🔹 Project 1 — Econometric Foundations & Linear Regression
Covers the foundations of financial econometrics including OLS regression, hypothesis testing, and model diagnostics applied to financial datasets.

**Key Topics:** Ordinary Least Squares (OLS), t-tests, F-tests, R-squared, residual diagnostics, heteroskedasticity, and autocorrelation tests.

---

### 🔹 Project 2 — Time-Series Analysis & Forecasting
Applies time-series econometric models to financial data including ARMA/ARIMA models, stationarity testing, and forecasting.

**Key Topics:** Stationarity (ADF/KPSS tests), ARMA/ARIMA modelling, Box-Jenkins methodology, model selection (AIC/BIC), and out-of-sample forecasting.

---

### 🔹 Project 3 — Volatility Modelling & Advanced Econometrics
Implements volatility modelling techniques and advanced econometric methods used in financial risk analysis.

**Key Topics:** ARCH/GARCH models, conditional heteroskedasticity, volatility clustering, Value-at-Risk (VaR), and cointegration analysis.
